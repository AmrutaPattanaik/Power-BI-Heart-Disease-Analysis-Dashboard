# Power-BI-Heart-Disease-Analysis-Dashboard
🫀 Heart Disease Analysis Dashboard – Power BI

An interactive and data-driven Power BI Dashboard designed to analyze heart disease patient data and uncover meaningful insights into survival patterns, age-wise trends, and gender-based differences (male vs female).

📌 Project Overview

This project transforms raw healthcare data into interactive visual insights, enabling users to explore how heart disease outcomes vary across different age groups and between genders.

The dashboard provides a deeper understanding of:

📊 Age-related risk patterns
🚻 Differences between male and female patients
❤️ Impact of key medical conditions on survival
📊 Key Insights & Features

🔹 KPI Dashboard Summary

Displays:
Survival Rate
Average Age of Survivors
Total Survivals
Total Deaths

🔹 Age-wise Analysis

Breaks down survival trends across different age groups
Shows how health indicators like:
Serum Creatinine
Ejection Fraction
vary with age

🔹 Gender-based Analysis (Male vs Female)

Enables comparison of survival patterns between male and female patients
Helps identify gender-specific risk trends

🔹 Combined Age & Gender Insights

Interactive visuals allow analysis of how age and gender together influence survival outcomes

🔹 Survival Trend Visualization

Line chart showing survival rate variation across age groups

🔹 Health Risk Factor Analysis

Ribbon chart visualizing the effect of:
Smoking
Diabetes
High Blood Pressure
Anaemia

🔹 Interactive Slicer

Filter the dashboard by gender (male/female) for customized insights

Tool: Microsoft Power BI
Data_Processing: Power Query
Visualizations:
  - KPI Cards
  - Line Charts
  - Ribbon Charts
  - Column Charts
  - Slicers

🚀 How to Run the Project

1. Download the .pbix file from this repository
2. Open the file using Microsoft Power BI Desktop
3. Explore insights based on age-wise and gender-wise analysis
